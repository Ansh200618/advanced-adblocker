# Advanced Ad Blocker

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Ansh200618/advanced-adblocker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd advanced-adblocker
   ```
3. Load the extension in Chrome:
   - Go to `chrome://extensions/`
   - Enable `Developer mode`
   - Click on `Load unpacked`
   - Select the project directory.

## Features
- Comprehensive ad blocking capabilities.
- Privacy protection with easy-to-use controls.
- Statistics dashboard to monitor blocking activity.

## Building from Source
1. Ensure you have Node.js installed.
2. Run:
   ```bash
   npm install
   ```
3. Build the extension:
   ```bash
   npm run build
   ```

## Usage Guide
- Use the popup interface to toggle blocking.
- Access settings for custom filter management.
