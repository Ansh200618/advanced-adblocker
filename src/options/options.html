About Advanced Ad Blocker

Version

1.0.0

Description

Advanced Ad Blocker is a comprehensive browser extension that blocks ads, trackers, and malicious scripts to provide a faster, safer, and more private browsing experience.

Features

  • ✓ Block ads across all websites
  • ✓ Prevent tracking and analytics
  • ✓ Cosmetic filtering to hide ad placeholders
  • ✓ Element picker to block specific page elements
  • ✓ Custom filter rules
  • ✓ Whitelist management
  • ✓ Detailed blocking statistics

Privacy

This extension does not collect, store, or transmit any personal data. All settings and statistics are stored locally on your device.
